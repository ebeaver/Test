# LLM 기반 RAG 챗봇 프로젝트

이 프로젝트는 Python 기반으로 **LLM(Language Model)을 활용한 RAG(Retrieval-Augmented Generation) 챗봇**을 구현한 실습용 저장소입니다.  
강사님이 제공한 기본 소스 코드(`rag_chatbot.py`, `memory.py`)를 분석∙이해한 뒤, 이를 기반으로 기능을 수정하고 확장하는 것을 목표로 합니다.

---

## 📌 프로젝트 목표

1. **Python 환경에서 LLM을 직접 활용해 챗봇 기능 구현**
2. 강사님 제공 기초 소스:
   - `rag_chatbot.py`
   - `memory.py`
   
   → 두 파일의 구조 및 동작 원리 분석
3. 기존 코드를 수정·보완하여 **추가 기능**을 직접 구현
4. FAISS 기반 벡터 검색, Memory 기능 등을 사용한 **RAG 챗봇 구조 이해**
